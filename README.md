IRCTweet
======

IRC Notification for Twitter

##How to USE

###1.Set Up Your Twitter API Key

```
$pub run bin/irctweet.dart
```

Please Input Your

* Twitter consumer_key
* Twitter consumer_sercret
* Twitter access_key
* Twitter access_sercret

###2.Start Notice Bot

```
$pub run bin/irctweet.dart
```

Input 

* irc host name
* bot nickname
* channel name
